# 🧾 Inventory & Billing System

A professional desktop application built using Python and Tkinter to manage inventory, sales, and billing efficiently.

---

## 🚀 Features

- 📦 Product Management (Add, Update, Delete)
- 🔍 Search Products
- 🛒 Cart System
- 💰 Billing with Invoice Generation (PDF)
- 👥 User Login System (Admin & Staff)
- 📊 Sales Reports (Daily, Monthly, Custom)
- 📈 Sales Graph Visualization
- 🧾 Activity Logs (CSV & PDF Export)
- 🖼️ Product Image Support
- 💾 Auto Backup System

---

## 🛠️ Tech Stack

- Python
- Tkinter (GUI)
- SQLite (Database)
- Matplotlib (Graphs)
- FPDF (PDF Generation)
